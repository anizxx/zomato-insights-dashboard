 Zomato Insights Dashboard

 Project Overview :

This project is a Power BI dashboard that provides insights into Zomato data, helping analyze customer behavior, restaurant trends, and business performance.

Objectives :

* Analyze restaurant distribution across locations
* Identify popular cuisines
* Understand customer ratings and preferences
* Visualize pricing and order trends

 🛠 Tools Used

* Microsoft Power BI
* Data Visualization Techniques
